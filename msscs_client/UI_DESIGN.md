# MSSCS Client UI Design

## Design Philosophy

**AMOLED Dark Theme** - Pure black backgrounds (#000000) for OLED displays with vibrant accent colors.

**Flat & Modern** - No shadows or gradients except for accents. Clean, minimal interface.

**Neon Accents** - Cyan-green gradient (#00ff88 â†’ #00ccff) for primary actions.

## Color Palette

```
Primary Background:   #000000 (Pure Black)
Secondary Background: #0a0a0a (Near Black)
Tertiary Background:  #0f0f0f (Dark Gray)
Border:               #1a1a1a (Subtle Gray)
Text Primary:         #ffffff (White)
Text Secondary:       #888888 (Gray)
Text Tertiary:        #666666 (Dark Gray)
Accent Primary:       #00ff88 (Neon Green)
Accent Secondary:     #00ccff (Cyan)
Success:              #00ff88
Warning:              #ffaa00
Error:                #ff0000
```

## Component Styles

### Buttons

**Primary Button (Gradient)**
```css
background: linear-gradient(135deg, #00ff88, #00ccff);
color: #000000;
padding: 10px 20px;
border-radius: 8px;
font-weight: 600;
```

**Secondary Button**
```css
background: #1a1a1a;
color: #ffffff;
padding: 10px 20px;
border-radius: 8px;
font-weight: 600;
```

### Cards

```css
background: #0f0f0f;
border: 1px solid #1a1a1a;
border-radius: 12px;
padding: 16px;
```

### Status Indicators

**Online:** Green dot with glow
```css
background: #00ff88;
box-shadow: 0 0 12px #00ff8860;
```

**Offline:** Gray dot
```css
background: #666666;
```

**Syncing:** Orange dot with glow
```css
background: #ffaa00;
box-shadow: 0 0 12px #ffaa0060;
```

## Typography

```
Title:     28px, 700 weight
Subtitle:  18px, 600 weight
Body:      14px, 500 weight
Caption:   12px, 400 weight
Label:     11px, 600 weight, uppercase
```

## Animations

**Hover Effects:**
- Buttons: `translateY(-2px)` + glow
- Cards: `translateY(-2px)` + border color change

**Loading:**
- Spinner: Rotating border animation
- Progress bars: Smooth width transition

**Transitions:**
- All: `0.2s ease` for smooth feel

## Accessibility

- High contrast ratios (WCAG AAA)
- Keyboard navigation support
- Focus indicators on all interactive elements
- Screen reader friendly labels

## Responsive Behavior

**Minimum Window Size:** 800x600
**Optimal Size:** 1200x800

**Sidebar:** Fixed 240px width
**Content:** Flexible, scrollable
**Grid:** Auto-fit columns (min 300px)
